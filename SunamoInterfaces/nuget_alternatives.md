# NuGet Alternatives to SunamoInterfaces

This document lists popular NuGet packages that provide similar functionality to SunamoInterfaces.

## Overview

Common interfaces and abstractions

## Alternative Packages

### System.Runtime
- **NuGet**: System.Runtime
- **Purpose**: Built-in BCL interfaces
- **Key Features**: IDisposable, IComparable, IEquatable, IEnumerable

### System.ComponentModel
- **NuGet**: System.ComponentModel
- **Purpose**: Component interfaces
- **Key Features**: INotifyPropertyChanged, IDataErrorInfo

### MediatR.Contracts
- **NuGet**: MediatR.Contracts
- **Purpose**: Mediator pattern interfaces
- **Key Features**: IRequest, INotification abstractions

## Comparison Notes

Most interface needs are covered by System.Runtime and System.ComponentModel.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **System.Runtime**: Built-in BCL interfaces
- **System.ComponentModel**: Component interfaces
- **MediatR.Contracts**: Mediator pattern interfaces
